EDiE DYN Simulator: User Manual

このシミュレーターは、進化・多様性・環境を軸にAIの可能性を探る研究プロジェクト EDiE (Evolutionary Discovery Engineering) の一環として開発されました。特に DYN (Dynamics & Resilience) トラックにおける「環境激変下での適応力」を検証するための主要ツールです。

1. コンセプト：急変耐性こそ正義 (Resilience as Justice)

従来の機械学習や最適化アルゴリズムの多くは、静的な環境（固定された地形）でいかに早く「谷底（最適解）」に到達するかを競います。しかし、現実世界では「前提」そのものが激変します。

本シミュレーターは、「安定した環境での最適解は、激動の時代においては脆弱性（Fragility）と同義である」という仮説に基づき、地形が激変した瞬間にいかに素早く立ち直り、新しい最適解を見つけ出せるか（急変耐性）を評価します。

2. 実行環境

このツールは以下の2つの方法で利用可能です。

オンライン・デモ (Cloud Run): こちらをクリックして起動

ローカル実行 (Processing):

Processing 4.x をインストール。

BallSimulation.pde を開き、実行ボタンをクリック。

3. 操作ガイド

Phase 0: セットアップ (設定画面)

起動直後は設定モードです。

変異率（Mutation Rate）の調整: マウスを左右に動かすことで、次世代に受け継がれる「学習率（LR）」の変動幅を調整できます。

シミュレーション開始: 画面をクリックすると、第1世代のサバイバルテストが始まります。

Phase 1: シミュレーション (生存テスト)

100個体のエージェント（ボール）が地形を転がり、最適解を探します。

赤いボール: 現在の世代で最も適合度が高い「エリート」個体です。

青/白のボール: 探索を行っている他の個体たちです。

世代交代: 8秒ごとに評価が行われ、最も優秀な個体の遺伝子（パラメータ）が次世代に引き継がれます。

Phase 2: 結果表示 (統計グラフ)

10世代のテストが終了すると、自動的に統計画面に切り替わります。

適応力の推移: 学習率（Learning Rate）が世代を経てどのように変化（進化）したかをグラフで表示します。

4. 特殊操作と「観察猶予」ロジック

キー/操作

アクション

研究上の意味

C キー

環境激変のトリガー

地形をランダムにスライドさせ、前提を崩壊させます。

マウスクリック

重力介入

カーソル位置に局所的な「引き」を作り、外乱への耐性をテストします。

レジリエンス・ウィンドウ (Resilience Window)

Cキーを押して地形を激変させた瞬間、評価タイマーは強制的に**4秒間（240フレーム）**に書き換えられます。この4秒間は、エージェントが新しい重力バランスの中で「体勢を立て直す」ための猶予期間です。この期間内に安定できなかった個体は、次世代に子孫を残すことができません。
